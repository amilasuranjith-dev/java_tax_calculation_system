# Tax Calculation System

A comprehensive Java console application for calculating various types of taxes with Sri Lankan tax regulations.

## Features
- **Multiple Tax Types**:
  - Withholding Tax (Rent, Bank Interest, Dividend)
  - Payable Tax (Employee payments)
  - Income Tax (Annual income)
  - SSCL Tax (Social Security Contribution Levy)
  - Leasing Payment Calculations

- **Professional Features**:
  - Progressive tax calculations
  - Multi-level menu navigation
  - Input validation
  - Clear console interface
  - Error handling

## Tax Categories

### 1. Withholding Tax
- Rent Tax (10% for rent ≥ 100,000)
- Bank Interest Tax (5%)
- Dividend Tax (14% for dividends ≥ 100,000)

### 2. Payable Tax
Progressive taxation based on monthly employee payments

### 3. Income Tax
Progressive taxation based on annual income

### 4. SSCL Tax
Combined Sales Tax (2.5%) + VAT (15%)

### 5. Leasing Payment
- Monthly installment calculation
- Lease amount finder
- Leasing category search

## How to Run
```bash
# Compile
javac TaxCalculator.java

# Run
java TaxCalculator
